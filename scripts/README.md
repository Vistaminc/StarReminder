# 📜 脚本说明

## 编译脚本

### 🚀 一键编译.bat（推荐）
**功能：** 完整编译整个项目
- 清理旧文件
- 编译ToastNotifier子程序
- 编译主程序
- 部署所有必需文件
- 验证部署结果

**使用：** 双击运行

---

### 🏃 编译并运行.bat
**功能：** 编译后直接运行程序
- 执行完整编译流程
- 自动启动StarReminder

**使用：** 双击运行

---

### 📦 发布Release版本.bat
**功能：** 生成可分发的Release版本
- Release模式编译
- 创建Release文件夹
- 复制所有必需文件
- 生成版本信息

**使用：** 双击运行
**输出：** `Release/` 文件夹

---

## 测试脚本

- 测试Defender样式通知（完整）
- 测试Defender样式通知（简洁）

### 👨‍💼 以管理员身份运行.bat

## 其他脚本

### 🔨 重新编译.bat

### 📷 convert-to-ico.ps1

## 推荐使用顺序

1. **首次编译：** `一键编译.bat`
2. **测试功能：** `测试通知.bat`
3. **日常开发：** `编译并运行.bat`
4. **发布版本：** `发布Release版本.bat`

---

## 注意事项

⚠️ **编译前提：**
- 已安装 .NET 8.0 SDK
- 在项目根目录执行

⚠️ **网络问题：**
- 如果出现 NU1900 警告，是网络问题
- 不影响编译和使用

⚠️ **路径问题：**
- 所有脚本都会自动切换到正确目录
- 可以从任何位置双击运行

---

**提示：** 遇到问题请查看 `../docs/编译问题解决.md`
